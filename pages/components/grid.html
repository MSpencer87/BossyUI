Basic Grid

The grid is set on the parent element and any children are assigned a default width. In this example each child element is 1/3 of the parent

{% highlight html linenos %}
1/3 Col
1/3 Col
1/3 Col
{% endhighlight %}
1/3 Col
1/3 Col
1/3 Col

Assigning Dynamic Widths

In order to assign columns of varying widths, simply add a "col-#" class to that element. Here we have a 2/3 column on the left and a 1/3 column on the right. If the col- value exceeds the parent value, ex: col-4, it will default to 100% width.

{% highlight html linenos %}
2/3 Col
1/3 Col
4/3 Col
{% endhighlight %}
2/3 Col
1/3 Col
4/3 Col

Responsive Grids

The base grid uses the "grid-" prefix, in order to adapt to wider screens you can add the "tablet-#" and "desktop-#" classes to the parent element which will change the grids base division unit as the screen increases in size.

{% highlight html linenos %}
2 Col
1 Col
1 Col
{% endhighlight %}
1/2 Desktop
1/3 Tablet
Full width Mobile

1/2 Mobile
